## Step to Setup the Bot!

> 1: Go To `.env` file and provide bot token & mongodb.
> Watch this video to know how to make mongodb url (you don't have to watch the whole video just watch it to get mongodb Atlas URL): https://youtu.be/0y3x8E7STng?si=qh8ayllas7fbOC6l

> 2: Open Terminal and Download Required Npm Package
> You can run this command to download all the packages (make sure you have node.js installed you can install it from here: https://nodejs.org/en)
- Copy & Paste this in terminal:
> npm install discord.js dotenv mongodb mongoose canvafy axios

> 3: Go to `config.json` file and setup channel ID's, Bot Admin ID's, Bot Prefix etc.

> 4: Open The Terminal and run `node .`.
